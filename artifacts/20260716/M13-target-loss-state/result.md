# M13 目标丢失状态机

## 结论

通过。新增ROS无关的跟踪状态机，状态仅包含`TRACKING`、`HOLDING`和`RETURNING`，
首版默认不自动扫描。

## 已验证行为

- 首次发现目标进入跟踪并请求清理PID历史。
- 目标丢失后在`hold_timeout`内保持当前位置。
- 到达超时边界后进入回中状态。
- 从保持或回中重新捕获目标时请求清理PID历史。
- 持续跟踪不会每周期重复清PID。
- 零保持超时会立即回中。
- 非法时间、时间倒退和非布尔可见状态被拒绝。

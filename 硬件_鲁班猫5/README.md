# 鲁班猫5 V1硬件参考资料

本目录保存项目实际板卡`Embedfire LubanCat-5 V1 / EBF410149V1R0`的厂商硬件资料，作为
Pin 32、PWM13_M1、40Pin 5V电源网络和后续硬件诊断的版本依据。资料由用户于2026-07-20
加入项目；PDF、DXF内容保持原样，不属于本项目生成物。

## 关键依据

- 硬件规格书第21-24页：DC 12V@2A输入、板卡供电参数和40Pin定义。
- 原理图第3页：`VCC5V0_SYS`电源树及40P GPIO共享电源关系。
- 原理图第4页：12V输入、MP8759 5V降压和`VCC5V0_SYS`。
- 原理图第41页：40Pin的Pin 2/4连接`VCC5V0_SYS`，Pin 32连接`GPIO4_B6 / PWM13_M1`。

完整文件哈希见`checksums.txt`。使用其他版本板卡时必须重新核对PCB编号、设备树和40Pin定义，
不能直接套用本目录结论。

# M15 SoC PWM与供电盘点（软件预检）

## 结论

软件部分通过，M15仍在进行中。鲁班猫5物理Pin 32的`GPIO4_B6 / PWM13_M1`设备树复用成功，
可以替代PCA9685完成首版单轴SG90信号输出。

## 验证结果

- 启动配置加载`rk3588-lubancat-pwm13-m1-overlay`。
- 新增`pwmchip3`，底层设备为`febf0010.pwm`，驱动为`rockchip-pwm`。
- PWM13_M1复用组明确对应Pin 142 `gpio4-14 / GPIO4_B6`。
- 无外设时成功配置20,000,000 ns周期和1,500,000 ns占空时间，即50 Hz中位信号。
- 启用期间内核显示`requested enabled`，Pin 142显示`function pwm13 group pwm13m1-pins`。
- 测试结束后已关闭并取消导出，没有遗留用户空间消费者。

取消导出后内核仍保留最后的周期、占空时间和PWM pinctrl选择，但通道未启用。这是驱动状态
保留，不表示仍在输出波形。

## 剩余验收

- 确认独立4.8–5V舵机电源及额定电流。
- 确认舵机电源负极、鲁班猫GND和信号地共地。
- 留存实际接线记录后才能完成M15并进入M16小步动作标定。
